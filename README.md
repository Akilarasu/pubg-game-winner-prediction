# PUBG Game Winner Prediction

## Project Overview
This project predicts the winning placement percentage of players in PUBG matches using machine learning models.

## Dataset
The dataset contains player statistics such as:
- kills
- damageDealt
- walkDistance
- weaponsAcquired
- matchType

Target Variable:
winPlacePerc

## Methodology
Steps followed in the project:

- Data preprocessing
- Exploratory data analysis
- Feature engineering
- Model training and comparison
- Model evaluation

## Models Used
- Linear Regression
- XGBoost
- LightGBM
